# Yazi 使用手册

`Yazi` 是一个用 Rust 编写的、极速的、基于终端的文件管理器。它采用非阻塞异步 I/O 设计，在处理大型目录或复杂任务时依然能保持丝滑的响应体验。

## 1. 核心特性

- **异步性能**：所有 CPU 密集型任务（如文件计数、图像解码）都分布在多个线程中，防止 UI 延迟。
- **图像预览**：原生支持多种图像协议（Kitty, Sixel, ITerm2）并集成了 `Überzug++`。
- **任务管理**：内置任务调度器，提供实时进度更新，并支持后台任务的取消或优先级调整。
- **高度可定制**：通过 TOML 文件进行高度配置，并支持 Lua 插件扩展。
- **多标签支持**：支持同时打开多个标签页管理不同目录。

## 2. 基础用法

### 2.1 启动 Yazi
在终端中输入以下命令启动：
```bash
yazi
```

### 2.2 启动并进入特定目录
```bash
yazi /path/to/directory
```

## 3. 核心快捷键

Yazi 默认使用类似 Vim 的快捷键。在应用内按 `~` 或 `F1` 可打开帮助菜单。

### 3.1 导航与视图
| 快捷键 | 说明 |
| :--- | :--- |
| `h` / `j` / `k` / `l` | 左（父目录）/ 下 / 上 / 右（进入目录/打开文件） |
| `.` | 切换隐藏文件的显示 |
| `q` | 退出程序 |
| `Q` | 退出并留在当前目录（需配置 Shell Wrapper） |

### 3.2 文件操作
| 快捷键 | 说明 |
| :--- | :--- |
| `y` | 复制 (Yank) |
| `x` | 剪切 (Cut) |
| `p` | 粘贴 (Paste) |
| `d` | 移至回收站 |
| `a` | 创建文件或目录（以 `/` 结尾表示目录） |
| `r` | 重命名 |
| `/` | 在当前目录下搜索 |
| `f` | 过滤当前视图 |

### 3.3 选择与标签
| 快捷键 | 说明 |
| :--- | :--- |
| `v` | 进入视觉模式（多选） |
| `Space` | 切换当前项的选择状态 |
| `t` | 新建标签页 |
| `w` | 关闭当前标签页 |
| `1-9` | 切换到指定标签页 |

## 4. 进阶配置

Yazi 的配置文件通常位于 `~/.config/yazi/`。

- **`yazi.toml`**：通用设置（排序、布局、打开方式）。
- **`keymap.toml`**：自定义快捷键。
- **`theme.toml`**：颜色和样式主题。

## 5. 实用技巧：退出后切换目录

为了让 shell 在退出 Yazi 时自动跳转到最后浏览的目录，建议在 `.bashrc` 或 `.zshrc` 中添加以下封装函数：

```bash
function y() {
	local tmp="$(mktemp -t "yazi-cwd.XXXXXX")"
	yazi "$@" --cwd-file="$tmp"
	if cwd="$(cat -- "$tmp")" && [ -n "$cwd" ] && [ "$cwd" != "$PWD" ]; then
		cd -- "$cwd"
	fi
	rm -f -- "$tmp"
}
```
之后，你可以使用 `y` 命令代替 `yazi` 来启动。
