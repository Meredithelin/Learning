## 简介

**PowerToys** 是 Microsoft 为 Windows 用户开发的一套免费系统实用工具集。它起源于 Windows 95 时代，现在已重新焕发新生，专为高级用户和追求效率的人群设计。

这套工具可以大幅提升你的 Windows 使用体验，涵盖窗口管理、文件搜索、键盘重映射、批量重命名、颜色选取等功能。

### 适用人群

- 经常需要多窗口并行工作的用户
- 开发者、设计师、内容创作者
- 追求键盘操作效率的用户
- 希望自定义 Windows 行为的进阶玩家

## 安装方法

### 方法一：GitHub 发布页（推荐）

1. 访问 PowerToys GitHub 发布页：
   https://github.com/microsoft/PowerToys/releases

2. 下载最新版本的 `.exe` 或 `.msi` 安装包

3. 双击运行安装程序，按提示完成安装

### 方法二：Microsoft Store

1. 打开 Microsoft Store
2. 搜索 "PowerToys"
3. 点击"获取"或"安装"

### 方法三：命令行安装（使用 winget）

```powershell
winget install Microsoft.PowerToys
```

### 安装后设置

安装完成后，PowerToys 会自动启动并在后台运行。你可以通过以下方式打开设置面板：

- 在任务栏托盘中找到 PowerToys 图标，双击打开
- 或使用快捷键（默认需手动配置）

## 核心工具详解

### 1. FancyZones — 窗口布局管理器

**用途**：将屏幕划分为自定义区域，快速将窗口"吸附"到指定位置。

**使用步骤**：

1. 打开 PowerToys 设置 → FancyZones
2. 点击"创建新布局"或选择预设模板
3. 调整区域大小和位置，保存布局
4. 按住 `Shift` 键拖动窗口到目标区域，窗口会自动吸附

**快捷键**：
- `Win + Shift + ~`：打开布局编辑器（可自定义）
- 拖动窗口时按住 Shift：激活区域吸附

**进阶技巧**：
- 可以创建多个布局，在不同场景切换使用
- 支持多显示器，每个屏幕可以有独立布局
- 可设置"闪过即吸附"，无需一直按住 Shift

---

### 2. PowerToys Run — 快速启动器

**用途**：类似 macOS Spotlight 或 Linux 的 Rofi，快速搜索和启动应用、文件、计算等。

**快捷键**：`Alt + Space`（默认）

**功能列表**：

| 功能 | 示例 |
|------|------|
| 启动应用 | 输入 "chrome" 打开 Chrome |
| 搜索文件 | 输入文件名或路径 |
| 计算器 | 输入 `123*45` 直接显示结果 |
| 打开网址 | 输入 URL 直接访问 |
| 系统命令 | 输入 "lock" 锁屏、"sleep" 睡眠 |
| 窗口切换 | 输入窗口名称快速切换 |
| 单位换算 | 输入 `100km to miles` |

**插件扩展**：
- 可启用/禁用各种插件（如 Windows 搜索、计算器、Shell 等）
- 可设置搜索范围、排除目录等

---

### 3. Keyboard Manager — 键盘重映射

**用途**：重新定义按键功能，创建自定义快捷键。

**两个主要功能**：

#### 键重映射（Remap keys）
- 将一个按键的功能替换为另一个按键
- 例如：将 Caps Lock 映射为 Ctrl 或 Esc

#### 快捷键重映射（Remap shortcuts）
- 将一个快捷键组合替换为另一个
- 例如：将 `Ctrl + C` 映射为 `Ctrl + Insert`

**操作步骤**：

1. 打开 Keyboard Manager 设置
2. 选择"重映射一个键"或"重映射快捷键"
3. 点击"+"添加新映射
4. 选择原键/快捷键 → 选择目标键/快捷键
5. 点击"确定"保存

**常用映射建议**：
- Caps Lock → Ctrl（程序员常用）
- Caps Lock → Esc（Vim 用户常用）
- 右 Alt → 左 Alt（适应不同键盘布局）

---

### 4. File Explorer 加载项

**用途**：增强 Windows 文件资源管理器功能。

**功能详情**：

#### 预览面板增强
- 支持 Markdown (.md) 文件预览
- 支持 SVG 图片预览
- 支持 PDF 预览
- 支持 G-code (.gcode) 文件预览
- 支持源代码文件预览（带语法高亮）

#### 缩略图增强
- SVG 文件显示缩略图
- PDF 文件显示首页缩略图
- HEIC/HEIF 图片格式支持

**使用方法**：

1. 打开文件资源管理器
2. 选择一个文件
3. 在右侧预览面板查看内容（需启用预览面板）

---

### 5. Image Resizer — 图片批量调整大小

**用途**：快速批量调整图片尺寸。

**使用方法**：

1. 在文件资源管理器中选中多张图片
2. 右键菜单中选择"调整图片大小"
3. 选择预设尺寸或自定义
4. 可选择是否保留原图、调整质量等

**预设尺寸**：
- 小（适合电子邮件）
- 中（适合网页）
- 大（适合桌面背景）
- 手机

**自定义选项**：
- 可添加自己的预设尺寸
- 可选择输出格式（保持原格式或转换为 JPEG/PNG）

---

### 6. PowerRename — 批量重命名工具

**用途**：使用正则表达式批量重命名文件。

**使用方法**：

1. 选中要重命名的文件
2. 右键菜单选择"PowerRename"
3. 输入搜索模式和替换内容
4. 可使用正则表达式进行复杂替换
5. 预览修改结果，点击"应用"

**正则表达式示例**：

| 目标 | 搜索模式 | 替换内容 |
|------|----------|----------|
| 替换空格为下划线 | `\s` | `_` |
| 添加前缀 | `^` | `prefix_` |
| 添加后缀 | `$` | `_suffix` |
| 删除数字 | `\d+` | （空） |
| 提取特定部分 | `(.*)(\d+)(.*)` | `$2_$1$3` |

---

### 7. Color Picker — 颏色选取器

**用途**：从屏幕任意位置选取颜色，获取颜色代码。

**快捷键**：`Win + Shift + C`（默认）

**使用方法**：

1. 按下快捷键激活
2. 移动鼠标到目标颜色位置
3. 点击确认选取
4. 颜色代码自动复制到剪贴板

**支持的颜色格式**：
- HEX：#RRGGBB 或 #RRGGBBAA
- RGB：rgb(255, 255, 255)
- HSL：hsl(360, 100%, 50%)
- HSV

**设置选项**：
- 可选择默认输出格式
- 可设置是否显示颜色历史

---

### 8. Shortcut Guide — 快捷键指南

**用途**：显示 Windows 和当前应用的快捷键列表。

**激活方式**：长按 `Win` 键（超过 200ms）

**显示内容**：
- Windows 系统快捷键列表
- 当前活动应用的快捷键（如果应用有注册快捷键）

**设置选项**：
- 可调整激活延迟时间
- 可选择是否覆盖原生 Win 键行为

---

### 9. Mouse Utils — 鼠标工具集

#### Find My Mouse（查找鼠标）
- **用途**：鼠标丢失时快速定位
- **激活方式**：按两次左 Ctrl 键
- **效果**：鼠标周围出现高亮圈

#### Mouse Highlighter（鼠标高亮）
- **用途**：演示时突出鼠标位置
- **快捷键**：`Win + Shift + H`
- **效果**：点击时显示高亮效果

#### Mouse Jump（鼠标跳转）
- **用途**：快速将鼠标跳到大屏幕另一位置
- **快捷键**：自定义
- **效果**：显示缩小版屏幕预览，点击跳转

---

### 10. Always On Top — 窀口置顶

**用途**：将任意窗口固定在最上层，不被其他窗口遮挡。

**快捷键**：`Win + Ctrl + T`（默认）

**使用方法**：

1. 选中目标窗口
2. 按下快捷键
3. 窗口标题栏会显示边框标识，表示已置顶
4. 再次按下快捷键取消置顶

**适用场景**：
- 视频播放窗口需要一直在前台
- 参考文档需要保持可见
- 监控工具需要持续观察

---

### 11. Awake — 防止系统休眠

**用途**：临时阻止 Windows 进入睡眠或锁屏状态。

**模式选项**：
- **无操作**：使用系统默认设置
- **保持唤醒一小时**：临时阻止休眠
- **保持唤醒（无限期）**：持续阻止，直到手动关闭

**界面状态**：
- 托盘图标显示当前状态
- 绿色表示保持唤醒，灰色表示使用默认

**适用场景**：
- 长时间下载或上传文件
- 远程会议期间
- 运行耗时任务

---

### 12. Screen Ruler — 屏幕测量工具

**用途**：测量屏幕上的距离、元素尺寸。

**快捷键**：`Win + Shift + M`

**测量模式**：
- 边界检测：自动检测元素边界
- 直线测量：手动绘制测量线
- 矩形测量：测量矩形区域

**单位**：像素 (px)

---

### 13. Quick Accent — 快速输入特殊字符

**用途**：快速输入带音标的字母和其他特殊字符。

**使用方法**：

1. 按住字母键（如 `e`）
2. 保持按住一段时间
3. 弹出特殊字符菜单
4. 按数字键或点击选择

**示例**：
- `e` → é, è, ê, ë, ē 等变体
- `a` → á, à, â, ä, ā 等变体
- 支持多种语言字符

---

### 14. Hosts File Editor — Hosts 文件编辑器

**用途**：安全便捷地编辑 Windows hosts 文件。

**位置**：`C:\Windows\System32\drivers\etc\hosts`

**功能**：
- 可视化编辑界面
- 启用/禁用条目
- 搜索过滤
- 自动备份

**适用场景**：
- 开发测试时映射域名到本地
- 屏蔽特定网站
- 配置内网域名解析

---

### 15. Registry Preview — 注册表预览

**用途**：预览 .reg 文件内容，安全导入注册表。

**功能**：
- 可视化显示注册表文件结构
- 预览将要添加/修改的键值
- 安全导入，避免误操作

---

### 16. Paste as Plain Text — 纯文本粘贴

**用途**：粘贴时去除所有格式，只保留纯文本。

**快捷键**：`Win + Ctrl + V`（可自定义）

**适用场景**：
- 从网页复制内容粘贴到文档时去除格式
- 复制代码时避免带入样式
- 保持目标文档格式一致

**快捷键**：

### 19. File Locksmith — 文件占用查看器

**用途**：查看哪个进程正在占用文件（无法删除/修改时使用）。

**使用方法**：

2. 显示占用该文件的所有进程
3. 可选择结束进程

**适用场景**：
- 文件删除失败时查找原因
- 文件被锁定无法编辑时解决冲突

- 粘贴为 Markdown

## 使用技巧与最佳实践

### 启动与运行管理

1. **自动启动**：PowerToys 默认开机自启，可在设置中调整
2. **资源占用**：现代版本优化良好，内存占用约 50-100MB
3. **更新**：建议保持更新，新版本经常添加有用功能

### 效率组合使用

**多窗口工作者**：
- FancyZones（窗口布局） + Always On Top（关键窗口置顶）
- 可快速建立高效的多窗口工作环境

**开发者/程序员**：
- Keyboard Manager（Caps Lock → Ctrl/Esc）
- PowerToys Run（快速启动终端、IDE）
- PowerRename（批量重命名代码文件）

**设计师**：
- Color Picker（颜色选取）
- Image Resizer（快速调整图片）
- FancyZones（设计软件布局）

**内容创作者**：
- Awake（长时间导出时保持唤醒）
- Advanced Paste（格式化粘贴）
- PowerToys Run（快速搜索素材）

### 性能优化建议

- 只启用你需要的工具，禁用不用的功能
- 定期清理 FancyZones 中不用的布局
- 如果快捷键冲突，及时调整

### 常见问题解决

**问题：快捷键不生效**
- 检查是否有其他软件占用相同快捷键
- 确认 PowerToys 正在运行
- 尝试重新映射快捷键

**问题：FancyZones 吸附不工作**
- 确认拖动时按住 Shift 键（或启用"闪过即吸附"）
- 检查目标窗口是否支持（某些 UWP 应用可能有问题）

**问题：PowerToys Run 搜索慢**
- 检查搜索范围，排除不必要的目录
- 禁用不需要的插件
