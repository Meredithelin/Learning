# Tmux 高频操作速查表 (Quick Reference)

这份手册是 `How to use tmux.md` 的精简补充版，旨在提供最常用命令的瞬间参考。

## 1. 核心前缀键 (The Prefix)
**`Ctrl + b`** (先按 `Ctrl + b`，松开，再按后续按键)

## 2. 会话管理 (外部终端执行)
| 命令 | 说明 |
| :--- | :--- |
| `tmux` | 启动一个新的会话 |
| `tmux new -s <name>` | 创建并命名新会话 |
| `tmux ls` | 列出所有正在运行的会话 |
| `tmux a` | 连接到最近的会话 (Attach) |
| `tmux a -t <name>` | 连接到指定名称的会话 |
| `tmux kill-session -t <name>` | 关闭指定会话 |

## 3. 内部常用快捷键 (Prefix + ...)
### 窗口与会话
- **`d`** - 分离当前会话 (Detach，回到外部终端，后台保留)
- **`c`** - 创建新窗口 (Create)
- **`w`** - 交互式选择窗口/会话 (极力推荐)
- **`,`** - 重命名当前窗口
- **`&`** - 关闭当前窗口

### 分屏 (Panes)
- **`%`** - **左右**垂直分屏
- **`"`** - **上下**水平分屏
- **`z`** - **最大化/还原**当前面板 (Zoom，查看大段日志神器)
- **`x`** - 关闭当前面板 (需确认)
- **`q`** - 显示面板编号，输入数字即可跳转
- **`方向键`** - 在面板间切换

### 复制与翻页
- **`[`** - 进入**复制模式** (可用方向键或 Vim 键翻页，`q` 退出)
- **`PageUp / PageDown`** - 快速翻页

## 4. 推荐配置补充 (.tmux.conf)
为了更符合现代操作习惯，建议在 `~/.tmux.conf` 中加入：
```tmux
# 开启鼠标支持 (点击切换、拖拽调整大小、滚轮翻页)
set -g mouse on

# 将分屏快捷键改为更直观的符号 (可选)
bind | split-window -h
bind - split-window -v
unbind '"'
unbind %

# 增加历史滚动行数
set -g history-limit 10000
```
